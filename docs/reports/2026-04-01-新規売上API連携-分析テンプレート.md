## 3. 判断ルール（どれを残す／止める）

### 広告セット単位

1. **KPI（例: コンバージョン単価 or ROAS）が明らかに良い広告セット** → **継続**（予算を寄せる候補）。
2. **費用がかさみ、コンバージョンがほぼゼロ** → **一時停止** または **予算削減** を検討。
3. **配信が極端に少ない**（インプレッションがほぼゼロ）→ 入札・予算・審査・学習の確認後に判断。

### 広告単位（同一広告セット内）

1. 同一セット内で **コンバージョン単価が悪い／成果ゼロ** の広告 → **停止**。
2. **上位1〜2本** に絞り、予算を集中（学習の分散を防ぐ）。

### 注意

- **3日程度**では揺れが大きい。**4/1時点は「第1ラウンドの切り分け」**と捉え、**1週間後に再評価**すると安全。
- コピーが同一で **クリエイティブ（画像・動画）だけ違う**場合は、**CTR・動画視聴・リンククリック単価**で素材の強弱を見る。
