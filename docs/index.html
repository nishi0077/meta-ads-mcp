30のツール

レポート確認、キャンペーン作成、予算変更、クリエイティブ管理まで。広告運用に必要な操作を網羅。

安全設計

書き込み操作はデフォルトでプレビューのみ。新規作成はすべて停止状態。意図しない配信を防止。

かんたん接続

Claude Desktop、Claude Code、Cursorに対応。MCP互換のクライアントならどれでも利用可能。

デフォルトで dry_run=true — 変更内容のプレビューだけを表示。
新規作成はすべて PAUSED 状態で作成されます。

Setup

3ステップで開始

01

インストール

terminal 
$ git clone https://github.com/nishi0077/meta-ads-mcp.git
$ cd meta-ads-mcp
$ python -m venv .venv && pip install -r requirements.txt
02

設定 & 認証

.env 
META_AUTH_TYPE=oauth
META_APP_ID=your_app_id
META_APP_SECRET=your_app_secret
03

接続

claude_desktop_config.json 
{
  "mcpServers": {
    "meta-ads": {
      "command": "python",
      "args": ["meta_ads_server.py"]
    }
  }
}

広告マネージャーを閉じよう。
